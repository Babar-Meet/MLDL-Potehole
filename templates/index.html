About This System

This application uses YOLOv8 (You Only Look Once version 8), a state-of-the-art object detection model, to identify potholes in road images. The model has been trained on a dataset of road images with pothole annotations.

Model: YOLOv8
Confidence Threshold: 0.25 (25%)
Supported Formats: JPG, JPEG, PNG, GIF, BMP, WebP
